What is event delegation in the context of DOM, and why is it useful?

Event delegation is a technique where a parent element manages events for child elements using a single listener. It improves efficiency by reducing the number of listeners and handles dynamically added elements automatically. This is achieved through event bubbling, allowing events to propagate from child to parent elements.
